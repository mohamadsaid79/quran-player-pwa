# 📖 قرآني | مشغل القرآن الكريم والإنتاجية (PWA)

تطبيق **قرآني** هو منصة ويب تقدمية (PWA) متكاملة تجمع بين الجانب الروحي والعملي؛ حيث يتيح لك الاستماع إلى سور القرآن الكريم بأصوات مشاهير القراء، مع إمكانية تنظيم مهامك اليومية ومتابعة إنتاجيتك باستخدام مؤقت تركيز (بومودورو) متطور. يتميز التطبيق بالعمل بالكامل دون الحاجة للاتصال بالإنترنت (Offline) وتصميمه العصري ذي الطابع الداكن والذهبي الأنيق.

---

## 🌟 مميزات التطبيق

### 1. 🎧 مشغل قرآن كريم متقدم
* **مؤثرات بصرية ديناميكية:** قرص دوار يدور أثناء التشغيل مع أمواج صوتية متحركة تتفاعل مع حالة الصوت.
* **تحكم كامل في التشغيل:** أزرار للتنقل بين السور (السابقة/التالية)، وتكرار السورة الحالية، وضبط سرعة التشغيل (من 1.0x وحتى 2.0x).
* **شريط تقدم تفاعلي:** لمشاهدة الوقت المنقضي والمتبقي وتسهيل تقديم وتأخير الصوت.
* **البحث السريع:** إمكانية البحث عن القراء المفضّلين لديك بشكل فوري.

### 2. ⚡ العمل دون اتصال بالإنترنت (PWA)
* **تطبيق ويب تقدمي:** قابل للتثبيت على جميع الأجهزة (الهواتف الذكية، الأجهزة اللوحية، والحواسيب المكتبية).
* **إدارة كاش متقدمة (Cache API):**
  * **تحميل كامل:** زر مخصص لتحميل جميع سور القارئ للاستماع إليها لاحقاً أوفلاين.
  * **الكاش التلقائي:** يتم تحميل السورة التي تستمع إليها حالياً تلقائياً في الخلفية لمرة واحدة، لتكون متاحة أوفلاين في المرات القادمة.
  * **إدارة المساحة:** خيار لحذف ملفات الكاش الخاصة بأي قارئ لتحرير مساحة تخزين الجهاز.
* **حالة الاتصال:** مؤشر تفاعلي يوضح ما إذا كنت متصلاً بالإنترنت (Online) أو تعمل في وضع عدم الاتصال (Offline).

### 3. ⏱️ مؤقت التركيز والإنتاجية (Pomodoro)
* **تصميم دائري أنيق:** حلقة تقدم تفاعلية تدريجية تبين الوقت المتبقي.
* **فترات محددة مسبقاً:** فترات تركيز سريعة (25 دقيقة) أو طويلة (45 دقيقة).
* **تخصيص الوقت:** إمكانية إدخال وقت مخصص بالدقائق وتطبيقه مباشرة.
* **التحكم بالوقت:** أزرار لبدء المؤقت، إيقافه مؤقتاً، أو إعادة ضبطه.

### 4. 📝 قائمة مهام مرتبطة بالقارئ (Reciter-Linked Todo List)
* **جدولة مخصصة:** ترتبط قائمة المهام بالشيخ/القارئ المحدد حالياً؛ مما يتيح لك ربط أهداف حفظ أو استماع أو مهام عمل معينة بقارئ محدد.
* **حفظ تلقائي:** يتم حفظ حالة المهام (مكتملة/غير مكتملة) تلقائياً لكل شيخ في ذاكرة المتصفح المحلية (`LocalStorage`).
* **إحصائيات ذكية:** عدّاد للمهام المتبقية وزر لمسح جميع المهام المكتملة دفعة واحدة.

### 5. 🛠️ لوحة تحكم لإدارة القراء (Admin Panel)
* **إضافة قارئ جديد:** إمكانية إضافة قارئ مخصص بكتابة اسمه، وصفه، الحرف التعريفي لأيقونته، ورابط سيرفر الـ MP3 الخاص به (مثل خوادم موقع MP3Quran).
* **إدارة القراء الحاليين:** لوحة لعرض وتعديل القراء أو حذفهم بالكامل مع خيار مسح ملفات الكاش الصوتية الخاصة بهم.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **الواجهة الأمامية (Frontend):** 
  * **HTML5:** بناء هيكلي منظم ومتوافق مع معايير السيو (SEO).
  * **CSS3:** واجهة مستخدم مبهرة تدعم Glassmorphism وتأثيرات الإضاءة الذهبية المتوهجة مع تجاوب كامل مع كافة الشاشات (Responsive Design). خطوط عربية مميزة مثل `Tajawal` و `Amiri`.
  * **Vanilla JavaScript (ES6+):** معالجة المنطق البرمجي، التحكم بالصوت، وتحديث الواجهة تفاعلياً بدون مكتبات خارجية.
* **تطبيق ويب تقدمي (PWA):**
  * **Service Workers:** لإدارة الطلبات البرمجية وتوفير العمل أوفلاين عن طريق استراتيجية `Stale-While-Revalidate` لملفات التطبيق الأساسية، و `Cache-First` للملفات الصوتية.
  * **Web App Manifest:** لتمكين تثبيت التطبيق على الشاشات الرئيسية للأجهزة كأنه تطبيق محلي.
* **التخزين (Storage):**
  * **LocalStorage:** لحفظ قائمة المهام وبيانات القراء المضافين.
  * **Cache Storage API:** لتخزين ملفات الصوت (MP3) والصور والخطوط للعمل دون إنترنت.

## 🚀 طريقة التشغيل والتركيب

التطبيق مبني بالكامل باستخدام لغات الويب الأساسية (Vanilla JS/CSS/HTML)، ولا يحتاج إلى عملية بناء (Build Step) أو تثبيت حزم خارجية.

### التشغيل محلياً:
لتتمكن من اختبار ميزات الـ PWA والـ Service Worker، يجب تشغيل التطبيق عبر خادم محلي (Local Server) وليس بفتح ملف الـ HTML مباشرة. يمكنك استخدام أحد الخيارات التالية:

1. **باستخدام VS Code (Live Server):**
   * قم بتثبيت إضافة **Live Server** في متصفح Visual Studio Code.
   * انقر بالزر الأيمن على ملف `index.html` واختر `Open with Live Server`.
2. **باستخدام Python:**
   * افتح الطرفية (Terminal) في مجلد المشروع ونفذ الأمر:
     ```bash
     python -m http.server 8000
     ```
   * افتح المتصفح واذهب إلى: `http://localhost:8000`
3. **باستخدام XAMPP / WampServer:**
   * ضع مجلد المشروع داخل مجلد `htdocs` أو `www`.
   * شغل خادم Apache وافتح المتصفح على: `http://localhost/quran-player`

### تثبيت التطبيق (PWA):
بمجرد تشغيل التطبيق على متصفح يدعم تطبيقات PWA (مثل Google Chrome أو Microsoft Edge) عبر بروتوكول آمن `https` أو `localhost` محلياً:
1. ستظهر لك أيقونة **تثبيت التطبيق** في الشريط العلوي (الهيدر).
2. انقر عليها لتثبيت التطبيق مباشرة على جهازك ليصبح متاحاً في قائمة التطبيقات ويعمل في نافذة مستقلة وبشكل أسرع وأوفلاين.
